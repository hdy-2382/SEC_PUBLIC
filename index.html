데이터 갱신

정식 갱신 방법: 본인 PC에서 data/raw/ 에 새 xlsx를 복사 → python scripts/build_dashboard_json.py 실행 → git add data/ && git commit -m "..." && git push. Pages가 1~2분 뒤 새 데이터를 반영합니다.

임시 미리보기: 아래 영역에 데이터를 붙여넣으면 현재 브라우저 세션에서만 적용됩니다. 새로고침 시 사라지며 저장소 데이터에는 반영되지 않습니다.

형식 (탭 또는 콤마 구분): 평가일 ⇥ 입실인원 ⇥ 주 평가내용 ⇥ 일일평가 ⇥ 일일에러 ⇥ 연속성공 ⇥ 비고

또는 에러 로그 데이터 (헤더 제외)도 같은 방식으로 추가 갱신할 수 있습니다.
